# Assignment

Write a Python function `to_list_of_pairs(dictionary)` that returns all key/value pairs as a list of pairs.
The order is unimportant.

For example,

```python
>>> to_list_of_pairs({})
[]

>>> to_list_of_pairs({'a': 1})
[('a', 1)]

>>> to_list_of_pairs({'a': 1, 'b': 2})
[('a', 1), ('b', 2)]
```
