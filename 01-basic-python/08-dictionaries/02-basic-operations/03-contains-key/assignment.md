# Assignment

Write a Python function `contains_key(dictionary, key)` that checks whether `dictionary` contains an entry whose key equals `key`.

For example,

```python
>>> contains_key({'a': 1}, 'a')
True

>>> contains_key({'a': 1}, 'b')
False

# 1 is a value, not a key, so False is the desired result
>>> contains_key({'a': 1}, 1)
False
```
